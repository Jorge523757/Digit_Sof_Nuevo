## 🚀 Cómo Iniciar

1. **Ejecutar servidor**: 
   ```bash
   python manage.py runserver
   ```

2. **Acceder como Administrador**:
   - Ir a: http://127.0.0.1:8000/auth/login/
   - Usuario: `admin`
   - Contraseña: `DigitSoft2025@Admin!`

3. **Acceder como Usuario Normal**:
   - Ir a: http://127.0.0.1:8000/auth/register/
   - Crear cuenta nueva
   - Iniciar sesión
